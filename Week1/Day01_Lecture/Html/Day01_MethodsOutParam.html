Day 01: Out Parameters

Out Parameters

Out parameters are special by reference parameters.
Here are the differences:
  • The variable passed to the method does NOT need to be initialized. When calling a method with a ref parameter, the variable must be initialized. That is not required for out parameters.
  • The method with out parameters is required to assign a value to the out parameter before returning.

Examples

When calling int.TryParse, you don't need to initialize the int parameter.
In this example, TryParse will return false which will be assigned to isGood. It will also set number to 0 because "Bob" is not a number. 
int number;
bool isGood = int.TryParse("Bob", out number);
Note: when calling a method that has an out parameter, you must use the out keyword.

In this example, the method is required to assign a value to the out parameter. If the variable superName is not assigned a value, then you'll get a build error. 

public static void BestSuperhero(out string superName)
{
    superName = "Batman";
}
