Day 02: Multiple Delimiters

Sometimes the CSV string has multiple delimiters that separate the data.
In this case, you need to pass a char[] to the Split method.

Examples

In this example, our data has 2 delimiters: | and ?. 
string supersCSV = "Thor|Captain America|Iron Man?Thanos";
char[] delimiters = new char[] {'|', '?'};
string[] supers = supersCSV.Split(delimiters);

If the CSV string has delimiters that are right next to each other (EX: "Hello;;Hi"), Split will return an empty entry for the spot between the adjacent delimiters.
To remove the empty entries from the array that is returned from Split, use the StringSplitOptions.RemoveEmptyEntries enum when calling Split. 

string supersCSV = "Thor|Captain America|Iron Man|?Batman?Superman?Wonder Woman?";
char[] delimiters = new char[] {'|', '?'};
string[] supers = supersCSV.Split(delimiters, StringSplitOptions.RemoveEmptyEntries);

Challenges!

Try it!

  1. Split the string (use multiple delimiters and handle empty entries).
  2. Print the array that is returned.

public static void Main(string[] args)
{
    string characterInfo = "Level:5;Health:90;Skills:DoubleJump||Dodge|Roll";
    char[] delimiters = new char[] {':',';','|'};
    string[] values = characterInfo.Split(delimiters, StringSplitOptions.RemoveEmptyEntries);
    foreach(string item in values)
        Console.WriteLine(item);
}
