Day 02: Splitting Strings

A common practice to store multiple values in one variable is to use a string and store each value inside the string.
This is called comma-separated values or CSV.
To keep the data separated inside the string, you use a char or string as a separator often called a delimiter.
Even though it is called comma-separated values, you are not required to use a common. You can use any character or series of characters that are not part of the data. For instance, if the data contains commas, then you can't use a comma as a delimiter.

Examples

The string class has a method called Split. There are many overloads so pick one that fits your scenario.
The Split method always returns a string array. 
string supersCSV = "Thor|Captain America|Iron Man";
char delimiter = '|';
string[] supers = supersCSV.Split(delimiter);
In this example, | is our delimiter that separates the data inside of supersCSV.
