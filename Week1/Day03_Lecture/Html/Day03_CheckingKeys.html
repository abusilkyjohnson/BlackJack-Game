Day 03: Dictionaries - Checking Keys

There are 2 ways to check if a Key is in the Dictionary:
  • ContainsKey(key)
  • TryGetValue(key, out value)
Both of these methods will return a bool (true or false) if the key is found. TryGetValue(key, out value) will also return the value associated with the key in the out parameter if the key is found.

Examples

ContainsKey

Dictionary<string,float> menu = new Dictionary<string,float>()  
{
    {"Hamburger", 1.99F},
    {"Cheesy Hamburger", 2.99F}
};  
bool isFound = menu.ContainsKey("Iced Tea");
Console.WriteLine(isFound);
This example will print false because "Iced Tea" is not in the dictionary.

TryGetValue

Dictionary<string,float> menu = new Dictionary<string,float>()  
{
    {"Hamburger", 1.99F},
    {"Cheesy Hamburger", 2.99F}
};  
string keyToFind = "Cheesy Hamburger";
bool isFound = menu.TryGetValue(keyToFind, out float price);
if(isFound)
    Console.WriteLine($"{keyToFind} costs {price:C2}");
This example will print Cheesy Hamburger costs $2.99 because it was in the dictionary.

Challenges!

Try it!

Use either ContainsKey or TryGetValue to find the amazonian key. Print out the result of calling either method. 

public static void Main(string[] args)
{
   Dictionary<string, float> starRanking = new Dictionary<string, float>();
   {
     {"Batman", 5.0F},
     {"Superman", 4.5F},
     {"Aquaman", 1.0F}
   };
   string amazonian = "Wonder Woman";
   if(starRanking.TryGetValue(amazonian, out float ranking))
   {
        Console.WriteLine($"{amazonian} has a ranking of {ranking}.");
   }
   else
   {
        Console.WriteLine($"{amazonian} was not found in the dictionary.");
   }
}
