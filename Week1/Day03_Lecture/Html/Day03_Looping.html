Day 03: Dictionaries - Looping

The BEST way to loop over a dictionary is to use the foreach loop.

The type of the foreach variable will be a KeyValuePair<TKey,TValue> where the TKey and TValue will match the types used for the Dictionary.

If you try to modify the Dictionary in a foreach loop, you will get an exception.

Examples

Loop over the Dictionary

foreach(KeyValuePair<string,float> menuItem in menu)
{
    //the key will be the name of the menu item (EX: Hamburger)
    //the value will be the price of the menu item (EX: 1.99)
    Console.WriteLine($"{menuItem.Key}: {menuItem.Value:C2}");
}

public static void Main(string[] args)
{
   Dictionary<string, float> starRanking = new Dictionary<string, float>();
   {
     {"Batman", 5.0F},

     {"Aquaman", 1.0F}
   };
   foreach(var keyValuePair in starRanking)
   {
        Console.WriteLine($"{keyValuePair.Key}: {keyValuePair.Value} stars");
   }
}
