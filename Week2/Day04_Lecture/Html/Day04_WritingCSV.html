Day 04: Writing CSV

Comma-separated values (CSV) files have data stored where the individual pieces of data are separated by a delimiter. A delimiter is a char (or string) that is used as a separator. Example: "Batman,Superman,Wonder Woman". In this example, ',' is the delimiter that separates the data.

When writing out CSV data to a file, be sure to use a delimiter to separate the data.

To write the CSV text file, use the StreamWriter class.

Examples

This example will write out the values of the best object in CSV format. 
Superhero best = new Superhero() { Name = "Batman", SecretIdentity = "Bruce Wayne", Power = Powers.Money };
char delimiter = ';';
//1. open the file.
using (StreamWriter sw = new StreamWriter(filePath))
{
    //2. write to the file
    sw.Write(best.Name);
    sw.Write(delimiter);
    sw.Write(best.SecretIdentity);
    sw.Write(delimiter);
    sw.Write(best.Power);
}//3. CLOSE THE FILE!!
Here is the output generated: 
Batman;Bruce Wayne;Money
